stringifySimpleFilter

  • stringifySimpleFilter(filterName: string, options?: Record<string, any> | any[]): string
    • Defined in src/filters.ts:22

    Stringifies a given filter with its options to ffmpeg's filter graph syntax.

    example
    const filter = stringifySimpleFilter('scale', [1280, 720]);
filter === 'scale=1280:720';
const filter = stringifySimpleFilter('loudnorm', { linear: true });
filter === 'loudnorm=linear=true';
const filter = stringifySimpleFilter('negate');
filter === 'negate';

    Parameters

    • filterName: string

      The filter's name, VideoFilter AudioFilter

    • Optional options: Record<string, any> | any[]

      The filter's options.

    Returns string
